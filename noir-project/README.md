# NOIR — Premium Streaming Service

Полнофункциональный, готовый к использованию интерфейс премиального потокового сервиса с кино и сериалами.

## 🎬 Описание

NOIR — это современный, высокополированный UI/UX сервис потокового кино на основе чистого HTML, CSS и JavaScript. Проект разработан с профессиональными стандартами качества и предназначен как для демонстрации, так и для использования в качестве основы для реальных проектов.

## 🔧 Кастомизация

### Добавить новый фильм/сериал

Отредактируйте объект `mediaData` в конце HTML файла:

```javascript
myfilm: {
  title: 'Название фильма',
  orig: 'Original Title',
  year: 2024,
  poster: 'https://url-к-постеру.jpg',
  rating: 8.5,
  age: '16+',
  duration: 120,
  genres: ['Жанр 1', 'Жанр 2'],
  desc: 'Описание фильма...',
  studio: 'Студия',
  language: 'English'
}
```

Затем используйте в карточке:
```html
<div class="card" onclick="playMovie('myfilm', 'Название')">
  <!-- содержимое карточки -->
</div>
```

### Изменить цветовую схему

В блоке `:root` в стилях измените переменные:

```css
:root {
  --accent: #8b5cf6;      /* Основной цвет */
  --accent2: #6d28d9;     /* Вторичный цвет */
  --gold: #f59e0b;        /* Рейтинги */
  --bg: #09090f;          /* Фон */
  /* ... остальные переменные */
}
```

### Добавить свои шрифты

Импортируйте из Google Fonts или добавьте локально, затем обновите:
```css
body {
  font-family: 'Ваш шрифт', system-ui, sans-serif;
}
```

## 🎥 Видеоплеер

Проект использует встроенный HTML5 видеоплеер с поддержкой:

## 🐛 Решение проблем

### Видео не воспроизводится
1. Проверьте, что вы на локальном сервере (http://, не file://)
2. Убедитесь, что браузер поддерживает HLS (все современные браузеры поддерживают)

### Стили не загружаются
Все стили встроены в HTML. Если видите неправильные цвета, очистите кэш браузера.

### Поиск не работает
Убедитесь, что JavaScript включён. Откройте консоль браузера для отладки.
