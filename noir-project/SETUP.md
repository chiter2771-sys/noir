# 🚀 NOIR Setup Guide

## Быстрая установка за 30 секунд

### Вариант 1: Открыть в браузере
```
1. Распакуйте архив
2. Откройте `src/noir_streaming.html` в Chrome/Firefox
3. Готово!
```

### Вариант 2: Локальный сервер (для видео)
```bash
# Перейдите в папку проекта
cd noir-project

# Python 3
python -m http.server 8000

# Или используйте Node.js
npx http-server

# Или для более старых версий Python
python -m SimpleHTTPServer 8000
```

Затем откройте в браузере:
```
http://localhost:8000/src/noir_streaming.html
```

## Системные требования

- **Браузер:** Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **OS:** Windows, macOS, Linux
- **Интернет:** Требуется для загрузки постеров и видео
- **ОЗУ:** Минимум 512 MB
- **Экран:** 320px шириной и выше

## Использование на мобильном

1. Запустите локальный сервер на компьютере
2. Узнайте IP адрес:
   ```bash
   # macOS/Linux
   ifconfig | grep "inet "
   
   # Windows
   ipconfig
   ```
3. На мобильном откройте: `http://ВАШ_IP:8000`

## Редактирование контента

Всё содержимое находится в одном HTML файле. Чтобы отредактировать:

1. Откройте `src/noir_streaming.html` в текстовом редакторе
2. Найдите разделы:
   - **Фильмы/Сериалы:** поищите `mediaData = {`
   - **Цвета:** поищите `:root {`
   - **Текст:** Ctrl+F по названию раздела

3. Сохраните и откройте в браузере (F5 для refresh)

### Пример: Добавить новый фильм

Найдите строку ~2400 с объектом `mediaData` и добавьте:

```javascript
myfavoritefilm: {
  title: 'Мой любимый фильм',
  orig: 'My Favorite Film',
  year: 2024,
  poster: 'https://example.com/poster.jpg',
  rating: 9.0,
  age: '12+',
  duration: 120,
  genres: ['Драма', 'Приключения'],
  desc: 'Описание фильма...',
  studio: 'Студия',
  language: 'English'
}
```

Затем в HTML добавьте карточку:
```html
<div class="card" onclick="playMovie('myfavoritefilm', 'Мой любимый фильм')">
  <img class="card-img" src="https://example.com/poster.jpg" alt="Мой фильм">
  <!-- ... остальное -->
</div>
```

## Использование собственного видео

По умолчанию используется тестовое видео от Mux.dev. Чтобы использовать своё:

1. Загрузите видео на сервер (YouTube, Vimeo, собственный хост)
2. Получите URL (должен быть CORS-friendly)
3. В коде найдите `#player-video` и измените `src`

Пример:
```html
<video id="player-video">
  <source src="https://ваш-сервер.com/видео.mp4" type="video/mp4">
</video>
```

## Проблемы и решения

### Видео не загружается
- Проверьте, что вы на **http://**, а не **file://**
- Попробуйте другой браузер
- Убедитесь, что интернет работает

### Картинки не загружаются
- Проверьте подключение к интернету
- Unsplash может быть недоступен в вашей стране (используйте VPN)
- Или замените все URL на локальные изображения

### Стили выглядят странно
- Очистите кэш браузера (Ctrl+Shift+Del)
- Попробуйте режим Incognito
- Перезагрузите браузер

### Поиск не работает
- Откройте консоль (F12) и проверьте на ошибки
- Убедитесь, что JavaScript включён
- Попробуйте в другом браузере

## Оптимизация

### Сжать размер файла
Удалите неиспользуемый контент из `mediaData`. Каждый фильм = ~300 байт.

### Ускорить загрузку видео
Используйте HLS вместо MP4:
- Меньше на 60% размер
- Адаптивное качество
- Быстрее загружается

### Уменьшить использование памяти
Ограничьте количество одновременно загруженных изображений:
- Лениво загружайте постеры (lazy loading)
- Используйте WebP вместо JPG
- Сжимайте изображения онлайн
